# “数智黎锦”小程序需求文档


## 1. 这个小程序是干嘛的？
这是一个**黎锦文化+海南黎族乡村旅游的数字助农小程序**——“游客”身份下，你可以在里面了解黎锦非遗文化、“云逛”黎族村寨、预约织锦体验/农家宴、购买黎锦文创；“村民”身份下，你除了了解黎锦非遗文化、“云逛”黎族村寨，还能报名学习政府开办的进阶课程、接单增收（接的单即为游客预约的织锦体验/农家宴、购买的黎锦文创），“政府”身份下，你除了了解黎锦非遗文化、“云逛”黎族村寨，还能查看旅游数据、村民反馈。总体而言，这个小程序最终帮黎族乡村通过文化+旅游实现增收。


## 2. 主要功能有哪些？
### 游客
#### 底部有三个按钮：
- **线路推荐**：浏览黎锦乡村旅游路线、筛选体验项目，获取村寨导览入口；
- **AR互动**：（开发中）扫描黎锦实物触发AR文化解读，查看纹样背后的民俗故事；
- **个人中心**：“游客”身份下管理预约订单、收藏内容、个人信息及客服咨询。


#### 首页（分成顶部、中部、底部 三个区域）
##### 顶部区域
###### 平台标识与功能入口
- 左侧显示标题“黎族文化体验·助农增收”，字体加粗醒目；
- 右侧并列设置**语言切换**（中/英/俄，默认中文）、**身份切换**（游客/村民/政府，默认选中“游客”）按钮；

###### 公告栏

#### 个人中心页（入口位于首页底部第三列）
- **权限管理区**：显示当前账号权限（如“市级文旅局管理员”），支持查看下属区县账号列表；
- **功能入口**：
  - “数据权限设置”：跳转至页面设置哪些区域数据可见（如“仅查看保亭县数据”）；
  - “政策草稿箱”：查看未发布的政策草稿；
  - “操作日志”：查看近期操作记录（如“12月10日 发布政策X”）；
  - “账号安全”：修改密码、绑定手机等设置。




## 3. 界面和使用感觉要求
### 界面样子：
- 必须用 **TDesign Weixin** 这个界面工具包做（参考：https://tdesign.tencent.com/miniprogram/getting-started）；
- 风格要和“钟山风景区”小程序统一：简洁、清晰，同时加黎锦元素（比如按钮加小菱形纹、边框用横条纹）。


### 使用感觉：
- 用起来要流畅：滑动列表、切页面不能卡；
- 点按钮、等预约结果、加载图片时，都要有提示（比如“加载中...”“预约成功！”）；
- 村民用的按钮要**放大**（比如“今日订单”按钮100rpx大），方便操作。


### 速度：
- 小程序打开速度要快（≤3秒）；
- 村寨全景图、商品图加载要快（用懒加载）。


### 安全：
- 微信登录过程要安全，不能泄露用户信息；
- 村民/政府的专属功能，要对应身份才能看（比如游客进不了数据看板）。


## 4. 关于文化内容的优化建议
### 目标：
让游客更容易看懂、喜欢黎锦文化。

### 怎么做：
- **知识概览输入/搜索提示**：搜索框里给例子，比如“试试搜：双凤朝阳纹 + 寓意”；
- **给灵感**：加个“热门文化”按钮，点了能看到“双面绣技艺”“哈方言纹样”等热门内容+对应图片；
- **常用标签按钮**：在搜索框下面放标签（比如“龙被”“纹面”“黄道婆”），点一下就能搜到对应内容。
